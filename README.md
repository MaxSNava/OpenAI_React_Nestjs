# Asistentes de Open AI
Creacion de una app WEB con la libreria de Open AI para asistentes virtuales

## Herramientas
- Frontend: React
- Backend: NestJS

## Instalacion
1. Clonar el repositorio
```bash
git clone https:...
```
2. Instalar las dependencias de la carpeta frontend y backend
```bash
cd frontend
npm install

cd ../backend
npm install
```
